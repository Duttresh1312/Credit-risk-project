1) This project is made using American express open kaggle competition data. 

   Link to download or use data from: https://www.kaggle.com/competitions/amex-default-prediction/overview

2) you can run Data_simpling part first, I have used 20% of the train_labels of original dataset and merged with train_data.
   
   If you want you can skip this step and use/split data as you are required.

3) Run credit-risk-project file which includes,

   I) Data preprocessing
  II) Feature Engineering
 III) Model training
  IV) Grid Search 
   V) Choosing best model

4) Last use creditrisk-project-strategy file, refer as you required.
